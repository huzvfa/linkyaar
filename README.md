# LinkYaar 🔗

**Pakistan's #1 UGC Creator Marketplace**

> Link with Creators

LinkYaar connects Pakistani brands with top content creators for UGC campaigns. Brands browse, hire, and pay creators — all in one place.

## Pages
- `/` — Homepage
- `/creators` — Browse all creators
- `/pricing` — Subscription plans
- `/signup` — Register as brand or creator
- `/about` — About LinkYaar

## Tech Stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS

## Deploy on Vercel

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/YOUR_USERNAME/linkyaar)
